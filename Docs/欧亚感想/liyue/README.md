# liyue
任何一种管理体系都需要具备完善的评估组件才可以构成，而构成这些评估组件都有一定的局限性，因为不同管理体系都是针对不同的人群和项目所构建而成的。每一个重大的决策和管理项目都需要写成一篇完善的总结文档展现出来才可以推广至其地方。参与或者监督这个工作的人，都要互相讨论所建立的管理体系是否有说服力，并且能够的解决其他人提出的问题才可以公之于众进行推广。
作为一个有组织的机构进行审计和评估，使用的文档需要遵守管理现场标准，需要对开发人员进行评估，对实验者进行跟踪才可以确定新建的管理体系是否完备，并且可以查漏补缺，进一步完善管理体系。也应该模拟各种社会场景，将各种突发情况备案进行体系的完善才可以让设计者证明所创建的管理体系很完善值得推广。
综上所述，就是新建的管理体系针对不同的任务群体都有不同社会虚拟场景的建设，并且可以这些事情都有新制定的条约去约束。
